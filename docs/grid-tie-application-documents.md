# เอกสารที่ลูกค้าต้องเตรียม — ขออนุญาตขนานไฟโซลาร์รูฟท็อป

> สำหรับขั้น GridTie ของแอป — เอกสารทั้งหมดในลิสต์นี้ **ฝั่งลูกค้าต้องเตรียม**
> ส่วน SLD / ใบ กว. วิศวกร / spec อุปกรณ์ → ผู้ติดตั้งจัดเตรียมให้
> อัพเดต: เมษายน 2026 · ตรวจสอบล่าสุดกับเว็บ MEA / PEA ก่อนใช้จริง

## ⚠️ ข้อควรระวัง

เอกสารทุกรายการต้องเป็น **ชื่อ-นามสกุลเดียวกัน** (ผู้ใช้ไฟฟ้า / เจ้าของมิเตอร์ / เจ้าของบ้าน)
ถ้าคนละชื่อกัน ต้องมี**หนังสือยินยอม**จากเจ้าของที่ถูกต้องแนบเพิ่มเติมทุกครั้ง

## รายการเอกสาร

| # | เอกสาร | จำเป็นเมื่อ | หมายเหตุ |
|---|--------|-----------|----------|
| 1 | สำเนาบัตรประชาชน | **ทุกกรณี** | ชื่อตรงกับผู้ใช้ไฟฟ้า · เซ็นรับรองสำเนาถูกต้อง |
| 2 | สำเนาทะเบียนบ้าน | **ทุกกรณี** | เลขที่บ้านตรงกับจุดติดตั้ง · เซ็นรับรองสำเนาถูกต้อง |
| 3 | บิลค่าไฟฟ้าล่าสุด | **ทุกกรณี** | ใช้ยืนยัน CA No. / หมายเลขผู้ใช้ไฟ |
| 4 | หนังสือยินยอมให้ใช้สถานที่ + สำเนาบัตร ปชช. เจ้าของที่ดิน | เจ้าของมิเตอร์ ≠ เจ้าของบ้าน / เช่าอยู่ | เจ้าของสถานที่เซ็นยินยอม |
| 5 | หนังสือมอบอำนาจ + สำเนาบัตร ปชช. ผู้รับมอบอำนาจ | ให้ผู้ติดตั้งดำเนินการแทน | ส่วนใหญ่ต้องใช้เพราะให้บริษัทยื่นเรื่องให้ |
